# Atlantic MultiDecadal Oscillation (AMO) Index (ea_data)

Atlantic MultiDecadal Oscillation (AMO) Index (ea_data)

## Usage

``` r
data(amo)
```

## Format

An `ea_data` object:

- data:

  Tibble:

  - `year`: Year

  - `month`: Month (1–12)

  - `value`: SSTA (sea surafe temperture anomaly)

- meta:

  Includes:

  - `data_type`: "Atlantic Multidecadal Oscillation Index"

  - `region`: "Northern Hemisphere (0-60N)"

  - `location_descriptor`: "North Atlantic"

  - `units`: ""

  - `source_citation`: "NOAA ,
    https://www1.ncdc.noaa.gov/pub/data/cmb/ersst/v5/index/ersst.v5.amo.dat"

  - `original_value_col`: "SSTA"
